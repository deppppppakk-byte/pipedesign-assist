# Scroll Blocker Website

This repository contains a static website for the Scroll Blocker Android app.

## Pages

- `index.html` - Home page
- `download.html` - Download page
- `privacy-policy.html` - Privacy policy
- `terms.html` - Terms and conditions
- `assets/style.css` - Website styling
- `assets/main.js` - Small version/display script

## Before sharing publicly

Replace these placeholders:

- `yourwebsite.com`
- `your-email@example.com`

Add your real signed release APK to the `downloads` folder using your own GitHub upload or hosting control panel.

Do not upload a debug APK. Use only a signed release APK.

## GitHub Pages setup

Open repository Settings, go to Pages, choose Deploy from a branch, select `main`, select `/ root`, and save.
